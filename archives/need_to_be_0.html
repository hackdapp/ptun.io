什么决定你的自我驱动力
概念认知
Publish Date:   2019-07-12
Read Count:  

见

什么决定我的自我驱动力？ 答案就是：正确的刚需。

感

举例来说，英语作为自己的一个知识短板，学这么多年感觉都快放弃了。一次次放下，一次次却又捡起来。想想这么多年来对它的重新规划与精力付出，完全是靠自己的坚持去做这件不知道这次能否最后成功的事情，而且中间还时不时因各种突发事件暂停了。
再比如，锻炼身体，基本每年都会做一个锻炼计划，因为那时刻满脑一个念头身体是革命的本钱。但念头已过，就会发现动力明显不足，加之运动过程中频频出现的第二个反面声音说服自己放弃。

感

综合之前的成长学习过程，发现这些行动起初就没有建立在一个刚需的基础之上，潜意识中告诉自己这件事完全是可有可无，不会对自己造成大的影响。

参照日常中的刷牙习惯，刷牙完全不需要靠坚持去做，因为你潜意识中明白它可以保持牙齿的健康，否则牙痛的感觉你应该有所体会。所以要让自己做成一件事，需要先从认知上建立刚需诉求，这样在大量的时间投入只后自然而然就演变成个人的一个优秀习惯。

行

确定目标；思考所要做的事情是否为刚需，深层次挖掘对它的认知；整个过程本身其实就是一次重塑自我的过程。就好比你锻炼器械时，需要下意识将注意力关注在发力的肌肉群体上一般。思维同样也是需要这样的锻炼。

— The End —
