「系统」窗口管理-Hammerspoon
日增一技 Mac技巧
Publish Date:   2019-06-11
Read Count:  

在日常办公环境，有时候在处理一些事务性问题时，需要在浏览器网页、写作工具或者其他应用之间，频繁切换来完成某一项工作。当系统打开的应用比较多的时候，你会发现来回切换的操作总有好么一些不舒服。

至于，如何来解决这个问题呢？ 对，那就是将多个窗口放在同一屏幕中。

但有人可能发现，当平铺的窗口稍多一些时，可能在调整窗口大小的操作上还是会浪费一些时间。那么，是否有一些更好方法，可以快速对窗口进行界面布局中呢？

有，而且还不少。比如: Magnet、Moon、Sizeup、Cinch、Hammerspoon、LayAuto等

但是，今天我要推荐的是：Hammerspoon。 为什么要推荐这款软件而不是其他的呢？因为优势在于其强悍的个性化插件定制，而窗口管理功能只是其中的一个插件而已。

功能特色

  • 丰富的插件资源
  • 可编程插件
  • 灵活定义快捷键
  • 详尽的接口文档

效果展示

软件配置

1. 安装本体

1） 下载[http://t.cn/RgWjXtX]并解压

2） 拖动 Hammerspoon.app 至 /Applications

3） 打开 “系统偏好设 - “安全性与隐私” - “辅助” 找到 “Hammerspoon.app”, 将其勾选。

2. 克隆并自定义配置
    > git clone https://github.com/ashfinal/awesome-hammerspoon.git ~/.hammerspoon~

软件快捷键

命令 说明
⌘ + R 激活窗口场景
ESC 或 Q 退出窗口管理
A 或 ⇧+H 右左移动
D 或 ⇧+L 右右移动
W 或 ⇧+K 向上移动
S 或 ⇧+J 向下移动
H 左半屏
L 右半屏
K 上半屏
J 下半屏
Y 左上屏
O 右上屏
U 左下屏
I 右下屏
F 全屏
C 居中
= 放大
- 缩小
left 左侧扩展屏
right 右侧扩展屏
up 上侧扩展屏
down 下侧扩展屏
space 下一个扩展屏
[ 撤销上一步操作
] 重做
` 光标居中

注: 软件本身可扩展性比较强，只要你有想法基本就可以通过编程实现，但前提是你可能需要学习一下lua语言。
