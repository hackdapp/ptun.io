Arduboy
硬件
Publish Date:   2019-07-02
Read Count:  

Arduboy是一款只有信用卡大小的微型游戏系统。可运行经典8位游戏，并可对现有开源的游戏进行二级改造。另外，Arduboy是对外开源的，你可以通过它学习编程并创建自己的游戏应用。

Get Started

  1. Install Arduino & Libraries
    C++ development platform for programming microcontrollers.
  2. Download Games
    Browse the community to find new games.
  3. Upload to Arduboy
    Use Arduino or try Arduboy Manager to upload new games.
